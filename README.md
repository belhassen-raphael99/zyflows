# Zyflows - Architecture Web & Automatisation

## Stack Technique
- **Framework** : React + Vite
- **Styling** : Tailwind CSS
- **Déploiement Hostinger** : Zyflows (zyflows.com)

## Déploiement
Ce projet utilise un script de Prerendering personnalisé basé sur Puppeteer pour générer des fichiers statiques optimisés pour le SEO.

Le domaine officiel est : **https://zyflows.com**

## Scripts
- `npm run dev` : Lance le serveur de développement.
- `npm run build` : Compile le projet, exécute le prerendering et prépare les fichiers pour la production.
